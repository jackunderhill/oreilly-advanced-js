Full-Page.js

This is an example site for Full-Page.js

Features

  • The aim of this library is to allow you to easily make each section of your site automatically fit the height and width of the users viewport
  • You can set a limit so that the content will never shrink passed a certain limit
  • The library allows the sections to take the natural size of their contents for mobile views

Contribute

If you have any ideas for improvement or just want to poke around the source code, head over to github to check it out.
